Rumble x WDK x Agents

Watching becomes funding.

Flow is an autonomous quadratic tipping agent built on top of Rumble's Tether WDK wallet stack. When a viewer hits 50% watch time, Flow fires a `$0.10 USD₮` auto-tip. Completion fires `$0.25`. Every tip joins a 24-hour round, quadratic scores are computed, and the pool wallet only settles after signing the exact allocation plan hash.

How It Works

Autonomous treasury management without a human treasurer.

Watch-time auto-tips

Rumble webhook events trigger viewer-configured auto-tip rules with per-day budgets and token preferences.

Canonical settlement planning

Each round produces a deterministic JSON plan containing allocations, bridge actions, signatures, and execution receipts.

Cross-network creator payouts

Polygon is the pool home chain. EVM creator payouts can route through bridge actions, while TRON, Bitcoin, and TON remain native payout destinations.

Portal + overlay + notifications

Creators configure splits, payout destinations, and overlay settings in the portal. Live events stream to OBS and settlement tx hashes queue for Telegram delivery.
